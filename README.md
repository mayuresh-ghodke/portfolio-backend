# Portfolio Backend

This is the **Spring Boot backend** for my personal portfolio website.  
It provides REST APIs to manage projects, skills, experiences, contact messages, and user authentication.

---

## üõ† Technology Stack

- **Backend:** Java, Spring Boot  
- **Database:** MySQL  
- **ORM:** Spring Data JPA (Hibernate)  
- **Security:** Spring Security (form-based login)  
- **Build Tool:** Maven

---

## Features

- Manage user profile and contact messages
- CRUD operations for projects and skills
- Secure authentication for the admin panel
- RESTful APIs for frontend consumption
- Connects to MySQL database
